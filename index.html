Pod-ing Docs

Pod-ing은 Falco 런타임 이벤트와 Cilium Hubble 네트워크 플로우를 결합하여 Kubernetes 내부 공격 체인을 실시간으로 탐지하는 보안 상관분석 플랫폼입니다.

빠른 설치
helm repo add poding https://yew0n12.github.io/capstone-poding/

helm install poding poding/poding \
  -n poding-system \
  --create-namespace

Pod-ing이 해결하는 문제

Kubernetes 환경에서는 컨테이너 내부 행위와 Pod 간 네트워크 흐름이 분리되어 관찰됩니다. Pod-ing은 이 두 관점을 하나로 묶어 단일 이벤트가 아니라 공격 진행 흐름을 추적합니다.

런타임 이벤트 수집

Falco를 통해 shell 실행, 민감 파일 접근, ServiceAccount token 접근 등 컨테이너 내부 행위를 수집합니다.

네트워크 플로우 수집

Cilium Hubble을 통해 Pod 간 통신, 외부 연결, namespace 간 이동 흐름을 네트워크 관점에서 확인합니다.

FSM 기반 상관분석

서로 다른 이벤트를 시간순으로 연결하여 공격이 어느 단계까지 진행됐는지 상태 기반으로 추적합니다.

시스템 흐름

Falco와 Hubble에서 수집된 이벤트는 Pod-ing 상관분석 엔진으로 전달되고, 분석 결과는 Grafana 대시보드와 알림 지표로 확인할 수 있습니다.
