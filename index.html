Pod-ing

Kubernetes security correlation platform that combines Falco runtime events and Cilium Hubble network flows to detect attack chains in real time.

Quick Install

helm repo add poding https://yew0n12.github.io/capstone-poding/
helm repo update
helm install poding poding/poding -n poding-system --create-namespace

What is Pod-ing?

  • Collects runtime security events from Falco
  • Collects network flow data from Cilium Hubble
  • Correlates heterogeneous events using an FSM-based detection engine
  • Visualizes attack propagation through observability components

Requirements

  • Kubernetes cluster
  • Helm 3
  • Falco
  • Cilium / Hubble
